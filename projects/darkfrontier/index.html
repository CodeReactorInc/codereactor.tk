What is Dark Frontier?

Dark Frontier is updater from Code Reactor to you embed it in your application, he uses XML in your internal API to make communications, requests and verify the version and exit code to say to main program the result of your operation

What the list of apps that can use Dark Frontier?

At moment, only Windows APPs but we have the plans to extend the support for Linux and UNIX API using C and GTK
