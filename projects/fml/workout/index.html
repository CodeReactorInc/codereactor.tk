What is Workout Interpreter?

Workout is Functional Markup Language interpreter for any platform that has support for UNIX API

How API he uses?

He don't uses a specific API since hasn't a FML API defined at moment

What is the difference from Sirius?

He have a standalone core and he are writted in C/C++

What is the difference between core and interpreter?

The core has the memory storage, implementation API and the code reader but the tools like Console printer are implemented in interpreter, you can use the core for embed FML in your project for plugins or mods for example
